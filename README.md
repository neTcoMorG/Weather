# **날씨모아**
![1](https://user-images.githubusercontent.com/24227385/221663763-e36ffacb-f89b-4316-9808-f7acf8455a3a.png)

## **소개**
이 프로젝트는 **openweather api**를 이용하여 전국의 날씨를 수집 및 제공하고 <br/>수집된 데이터를 바탕으로 날씨 순위 정보를 제공합니다.
<br/>
<br/>

## 🚀 **사용한 기술**
### **BackEnd**
- Java
- Spring Boot
- JPA
- H2
- Thymeleaf
### **FrontEnd**
- HTML
- CSS
